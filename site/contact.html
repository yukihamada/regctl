よくあるご質問

regctl Cloud の料金体系を教えてください +
regctl Cloud はポイント制の従量課金です。ドメイン管理費は1ドメインあたり月額10ポイント（¥10）、ドメイン登録・更新費用は各レジストラーの料金に準拠します。初回は¥1,000〜の任意の金額でチャージが可能です。
Claude Code との連携方法を教えてください +
環境変数 REGCTL_AUTO_CONFIRM="true" および REGCTL_OUTPUT="json" を設定することで、Claude Code での自動化に最適化されます。確認なし実行（-y フラグ）とJSON出力により、AI開発ワークフローにシームレスに統合できます。
対応しているドメインレジストラーは何ですか？ +
現在、VALUE-DOMAIN、AWS Route 53、Porkbun に対応しています。統一されたAPIインターフェースにより、複数のレジストラーを一つのCLIで管理できます。追加のレジストラー対応も継続的に開発中です。
API の利用制限はありますか？ +
適正利用の範囲内であれば、API利用に特別な制限はありません。レート制限（1秒あたり10リクエスト）が適用されますが、一般的な利用では問題になりません。大量リクエストが必要な場合はエンタープライズプランをご検討ください。
データのセキュリティ対策について教えてください +
すべてのデータは暗号化され、Cloudflare Workers上で処理されます。OAuth認証、PBKDF2によるパスワードハッシュ化、JWTベースの認証システムを採用。監査ログによる完全な操作履歴管理も行っています。
無料トライアルはありますか？ +
アカウント作成は無料で、CLIのインストールと基本機能の確認が可能です。実際のドメイン管理には最低¥1,000のチャージが必要ですが、少額から始められます。まずは無料でCLIをお試しください。
原価なのにどうやって利益を出すの？ +
ドメイン登録・更新料はレジストラ原価（NameSilo、Namecheap等）をそのまま横流しし、利益はゼロ円です。月額10ポイント（¥10）の管理費のみで運営コストをカバーしています。この透明なビジネスモデルにより、ユーザーは業界最安値でドメインを管理できます。
原価は為替変動で変わりますか？ +
はい、為替レートとレジストラの価格改定に連動してリアルタイムで変動します。ダッシュボードで「本日の原価レート」を常に確認できます。価格変動はすべて利用規約に基づき自動的に転嫁されますが、管理費（10pt/月）は固定です。
